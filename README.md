# Пересдача дисциплины "Node.JS" за 3 курс I семестр

## Описание

Приложение представляет собой утилиту для получения новостей по интересующим его рубрикам с rss-лент новостного сайта "Ведомости" в формате pdf.

## Требования

Приложение состоит из трех подсистем:

- консольное взаимодействие,
- парсинг,
- рендеринг.

#### Консольное взаимодействие

Для реализации консольного взаимодействия используется библиотека `Inquirer.js`.

При первичном запуске пользователь выбирает интересующие его рубрики, они сохраняются в файле `storage/settings.json` следующего вида:

```json
{
  "selectedRubrics": ["business-energy", "politics"],
}
```

После настройки запускается фоновый режим.

#### Парсинг

Парсинг новостей происходит на основе рубрик из `storage/settings.json` и ссылок на их ленты в `storage/rubrics.json`.

#### Рендеринг

Результаты выгружаются по выбранному пользователем расписанию в `pdf`-файл с названием вида `vedomosti_yyyy_mm_dd_hh.pdf`. Для формирования отчета первично используется шаблонизатор `ejs`, который рендерит полученные данные в `html`, а сохранение в `pdf` происходит посредством `puppeteer`.
