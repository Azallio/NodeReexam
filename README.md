# Пересдача дисциплины "Node.JS" за 3 курс I семестр

Приложение представляет собой утилиту для получения новостей по интересующим его рубрикам с rss-лент новостного сайта "Ведомости" в формате pdf.

## Функциональные требования

1. При запуске пользователь выбирает интересующие его рубрики через терминал.
2. На основании выбранных рубрик и ссылок на их ленты из `storage/rubrics.json` происходит загрузка и парсинг новостей.
3. Результаты выгружаются в `pdf`-файл с названием вида `vedomosti_yyyy_mm_dd_hh.pdf`. Для формирования отчета первично используется шаблонизатор `ejs`, который рендерит полученные данные в `html`, а сохранение в `pdf` происходит посредством `puppeteer`.

## Задание и критерии оценивания

1.1 В функции `loadRubrics` модуля `src/storage.ts` осуществлено чтение файла с рубриками (1 балл), обработаны исключения (1 балл).

2.1 В функции `getUserInput` модуля `src/cli.ts` реализован выбор рубрик из числа представленных в файле `_rubrics.json` (1 балл).

3.1 В функции `loadNewsByRubric` модуля `src/loader.ts` реализовано получение новостей по рубрике (1 балл).

3.2 В функции `parseNewsByRubric` модуля `src/loader.ts` реализован парсинг полученных новостей из `XML` (2 балла).

4.1 В функции `renderMarkup` модуля `src/renderer.ts` осуществлен рендеринг html на основе шаблона с использованием `ejs` (2 балла).

4.2 В функции `generateFilename` модуля `src/renderer.ts` реализована генерация имени файла согласно формату (1 балл).

4.3 В функции `save` модуля `src/renderer.ts` реализовано сохранение pdf-файла на основании html-разметки с использованием `puppeteer` (1 балл).

_Максимальный балл по критериям - 10. Итоговый балл за пересдачу будет скорректирован множителем, который зависит от максимального доступного вам балла за пересдачу._
